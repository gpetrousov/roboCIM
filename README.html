roboCIM chessplayer

Description

The roboCIM chessplayer is a project that allows you to play chess or solve chess puzzles using the robotic arm Lab-Volt Servo Robot System, Model 5250 using our custom made platform. Our platform uses an arduino to send binary signals that correspond to certain commands-action that the arm will execute. The signals from the arduino are TTL inputs to the arm.
